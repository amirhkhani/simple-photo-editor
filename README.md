# simple-photo-editor
a very simple photo editor coded by python (PyQt)
<br></br>
you can open, rotate, flirp and half-size your photo easily but you will not have advanced Tolls!
<br></br> 
PyQt6 package has used for this mini-project
<br></br>

## how to use
1. `git clone https://github.com/amirhkhani/simple-photo-editor.git`
2. `cd simple-photo-editor`
3. `pip install -r requirements.txt`
4. `python src/photo_editor.py`
